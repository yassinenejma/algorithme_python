Les structures itératives en Algorithme et Python

Introduction aux structures itératives

Les structures itératives, également appelées boucles, sont des éléments fondamentaux en algorithmique et en programmation. Elles permettent d'exécuter plusieurs fois un même bloc d'instructions, ce qui est essentiel pour automatiser des tâches répétitives.

Définition : Une structure itérative (ou boucle) est une séquence d'instructions qui est exécutée plusieurs fois jusqu'à ce qu'une condition particulière soit atteinte.

Il existe trois types principaux de structures itératives :

  • La boucle POUR (for) : utilisée lorsque le nombre d'itérations est connu à l'avance
  • La boucle TANT QUE (while) : utilisée lorsque le nombre d'itérations est inconnu et dépend d'une condition
  • La boucle RÉPÉTER...JUSQU'À (do...while) : similaire à TANT QUE, mais garantit au moins une exécution

1. La boucle POUR (for)

La boucle POUR est utilisée lorsque vous connaissez à l'avance le nombre exact de fois que vous souhaitez exécuter un bloc d'instructions. Elle est particulièrement adaptée aux parcours de collections comme les listes, les chaînes de caractères ou les plages de nombres.

Syntaxe en pseudo-code (algorithme)

POUR variable DE valeur_début À valeur_fin [PAR pas] FAIRE
    instructions
FIN POUR

Note : Le paramètre pas est optionnel et vaut 1 par défaut.

Syntaxe en Python

En Python, la boucle for est généralement utilisée avec la fonction range() pour les séquences numériques :

# Syntaxe de base
for variable in range(début, fin, pas):
    # instructions
            
# Sans spécifier le début (commence à 0)
for variable in range(fin):
    # instructions

Attention : En Python, range(début, fin) génère des nombres de début à fin-1.

Exemples en pseudo-code et Python

Exemple 1: Afficher les nombres de 1 à 5

Pseudo-code :

POUR i DE 1 À 5 FAIRE
    AFFICHER i
FIN POUR

Python :

for i in range(1, 6):
    print(i)

Résultat : 1 2 3 4 5

Exemple 2: Calculer la somme des nombres de 1 à 10

Pseudo-code :

somme ← 0
POUR i DE 1 À 10 FAIRE
    somme ← somme + i
FIN POUR
AFFICHER somme

Python :

somme = 0
for i in range(1, 11):
    somme += i
print(somme)  # Affiche 55

Exemple 3: Parcourir une chaîne de caractères

Python :

mot = "Python"
for lettre in mot:
    print(lettre)

Résultat : P y t h o n (chaque lettre sur une ligne différente)

2. La boucle TANT QUE (while)

La boucle TANT QUE exécute un bloc d'instructions tant qu'une condition spécifiée est vraie. Contrairement à la boucle POUR, le nombre d'itérations n'est pas connu à l'avance et dépend de l'évolution de la condition au cours de l'exécution.

Important : Si la condition est fausse dès le départ, le bloc d'instructions ne sera jamais exécuté.

Syntaxe en pseudo-code (algorithme)

TANT QUE condition FAIRE
    instructions
FIN TANT QUE

Syntaxe en Python

while condition:
    # instructions

Exemples en pseudo-code et Python

Exemple 1: Compter jusqu'à 5

Pseudo-code :

compteur ← 1
TANT QUE compteur <= 5 FAIRE
    AFFICHER compteur
    compteur ← compteur + 1
FIN TANT QUE

Python :

compteur = 1
while compteur <= 5:
    print(compteur)
    compteur += 1

Résultat : 1 2 3 4 5

Exemple 2: Demander un nombre jusqu'à ce qu'il soit positif

Pseudo-code :

nombre ← -1
TANT QUE nombre < 0 FAIRE
    AFFICHER "Entrez un nombre positif:"
    LIRE nombre
FIN TANT QUE
AFFICHER "Merci, vous avez entré:", nombre

Python :

nombre = -1
while nombre < 0:
    nombre = int(input("Entrez un nombre positif: "))
print(f"Merci, vous avez entré: {nombre}")

Cas pratique : La boucle TANT QUE est particulièrement utile lorsque vous voulez continuer une action jusqu'à ce qu'une condition soit satisfaite, comme la validation d'une entrée utilisateur ou l'attente d'un événement.

3. La boucle RÉPÉTER...JUSQU'À (do...while)

La boucle RÉPÉTER...JUSQU'À est similaire à la boucle TANT QUE, mais avec une différence cruciale : le test de la condition s'effectue à la fin de la boucle, ce qui garantit que le bloc d'instructions est exécuté au moins une fois.

Syntaxe en pseudo-code (algorithme)

RÉPÉTER
    instructions
JUSQU'À condition

Attention : La boucle s'arrête lorsque la condition devient VRAIE (contrairement à TANT QUE qui continue tant que la condition est vraie).

Syntaxe en Python

Python ne possède pas de structure équivalente directe à RÉPÉTER...JUSQU'À, mais on peut la simuler :

# Méthode 1: avec while et une exécution garantie
while True:
    # instructions
    if condition:
        break

# Méthode 2: avec un while et une première exécution forcée
instructions
while not condition:
    instructions

Exemples en pseudo-code et Python

Exemple 1: Demander un mot de passe jusqu'à ce qu'il soit correct

Pseudo-code :

RÉPÉTER
    AFFICHER "Entrez le mot de passe:"
    LIRE mot_de_passe
JUSQU'À mot_de_passe = "secret"
AFFICHER "Accès autorisé"

Python :

while True:
    mot_de_passe = input("Entrez le mot de passe: ")
    if mot_de_passe == "secret":
        break
print("Accès autorisé")

Exemple 2: Jeu de devinette

Pseudo-code :

nombre_secret ← 42
RÉPÉTER
    AFFICHER "Devinez le nombre (entre 1 et 100):"
    LIRE proposition
    SI proposition < nombre_secret ALORS
        AFFICHER "Trop petit!"
    SINON SI proposition > nombre_secret ALORS
        AFFICHER "Trop grand!"
    FIN SI
JUSQU'À proposition = nombre_secret
AFFICHER "Bravo, vous avez trouvé!"

Python :

nombre_secret = 42

while True:
    proposition = int(input("Devinez le nombre (entre 1 et 100): "))
    
    if proposition < nombre_secret:
        print("Trop petit!")
    elif proposition > nombre_secret:
        print("Trop grand!")
    else:
        break
        
print("Bravo, vous avez trouvé!")

Points importants et pièges à éviter

Boucles infinies

Une boucle infinie est une boucle qui ne s'arrête jamais car sa condition de sortie n'est jamais atteinte. C'est généralement dû à une erreur de programmation.

Exemple de boucle infinie en TANT QUE :

compteur = 1
while compteur > 0:
    print(compteur)
    compteur += 1  # Le compteur augmente toujours, 
                   # donc la condition reste toujours vraie

Solution :

compteur = 1
while compteur < 10:  # Ajouter une condition de sortie
    print(compteur)
    compteur += 1

Initialisation des variables

Toujours initialiser vos variables avant d'entrer dans une boucle, surtout pour les boucles TANT QUE.

Incrémentation dans les boucles TANT QUE

N'oubliez pas de mettre à jour les variables utilisées dans la condition de la boucle TANT QUE, sinon vous risquez de créer une boucle infinie.

La structure break et continue en Python

Python offre deux instructions spéciales pour contrôler le flux des boucles :

  • break : sort immédiatement de la boucle
  • continue : passe directement à l'itération suivante

Exemple avec break :

for i in range(1, 11):
    if i == 5:
        break  # Sort de la boucle quand i vaut 5
    print(i)
# Affiche: 1 2 3 4

Exemple avec continue :

for i in range(1, 6):
    if i == 3:
        continue  # Saute à l'itération suivante quand i vaut 3
    print(i)
# Affiche: 1 2 4 5

Exercices pratiques

Exercice 1: Somme des nombres pairs

Écrivez un programme qui calcule la somme des nombres pairs entre 1 et 100 en utilisant une boucle FOR.

Conseil: Un nombre est pair si le reste de sa division par 2 est égal à 0 (n % 2 == 0).

Exercice 2: Table de multiplication

Écrivez un programme qui affiche la table de multiplication d'un nombre saisi par l'utilisateur (de 1 à 10).

Exemple pour le nombre 7:

7 x 1 = 7
7 x 2 = 14
...
7 x 10 = 70

Exercice 3: Validation d'entrée

Écrivez un programme qui demande à l'utilisateur d'entrer un nombre entre 1 et 10. Si l'utilisateur entre un nombre en dehors de cette plage, le programme doit continuer à demander jusqu'à ce qu'un nombre valide soit entré.

Utilisez une boucle TANT QUE pour cet exercice.

Exercice 4: Factorielle

Écrivez un programme qui calcule la factorielle d'un nombre n saisi par l'utilisateur.

Rappel: La factorielle de n (notée n!) est le produit de tous les entiers positifs inférieurs ou égaux à n.

Exemple: 5! = 5 × 4 × 3 × 2 × 1 = 120

Exercice 5: Nombres premiers

Écrivez un programme qui vérifie si un nombre saisi par l'utilisateur est premier ou non.

Rappel: Un nombre premier est un nombre entier supérieur à 1 qui n'est divisible que par 1 et par lui-même.

Utilisez une boucle FOR et la structure break si nécessaire.

Exercice 6: Suite de Fibonacci

Écrivez un programme qui affiche les n premiers termes de la suite de Fibonacci, où n est saisi par l'utilisateur.

Rappel: La suite de Fibonacci commence par 0 et 1, puis chaque terme suivant est la somme des deux termes précédents.

Exemple: 0, 1, 1, 2, 3, 5, 8, 13, 21, ...

Conclusion

Les structures itératives sont des outils puissants en programmation qui vous permettent d'automatiser des tâches répétitives. Chaque type de boucle (POUR, TANT QUE, RÉPÉTER...JUSQU'À) a ses propres cas d'utilisation et avantages.

Conseils pour bien choisir votre boucle :

  • Utilisez une boucle POUR lorsque vous connaissez le nombre d'itérations à l'avance.
  • Utilisez une boucle TANT QUE lorsque vous ne savez pas combien d'itérations seront nécessaires.
  • Utilisez une boucle RÉPÉTER...JUSQU'À (simulée en Python) lorsque vous voulez garantir au moins une exécution.

La maîtrise des structures itératives est fondamentale pour tout programmeur. Avec de la pratique, vous deviendrez plus à l'aise pour choisir la bonne structure pour chaque situation.
